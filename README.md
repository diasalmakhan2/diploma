# mobileappDiploma

Офлайн Flutter-приложение для дипломной демонстрации: регистрация, вход, 3 урока с тестами, подсчет очков и профиль.

## Что уже реализовано

- Регистрация и вход студента
- Главный экран с 3 готовыми уроками
- 2 видео-урока и 1 reading-урок
- 10 вопросов в каждом тесте
- Начисление 1 очка за каждый правильный ответ
- Экран банка очков
- Профиль и выход из аккаунта
- Локальное хранение данных через `shared_preferences`

## Почему без Firebase

Для показа комиссии лучше офлайн-режим: приложение работает без интернета, без настройки Google Console и без риска, что авторизация или база не откроются на чужом телефоне.

Firebase можно подключить позже, если понадобится:

- облачное хранение аккаунтов
- синхронизация прогресса между устройствами
- настоящая регистрация/авторизация

## Как запустить

1. Установить Flutter SDK.
2. В корне проекта выполнить:

```bash
flutter create .
flutter pub get
flutter run
```

Команда `flutter create .` сгенерирует недостающие платформенные папки `android`, `ios`, `web` и служебные файлы.

## Как добавить видео

Положите локальные `.mp4`-файлы сюда:

- `assets/videos/daily_routine.mp4`
- `assets/videos/trip_to_london.mp4`

После этого выполните:

```bash
flutter pub get
flutter run
```

Если видеофайлы еще не добавлены, приложение покажет заглушку вместо плеера.
